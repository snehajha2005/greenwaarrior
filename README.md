# 🌱 Green Warrior

Green Warrior is an eco-friendly web platform designed to spread environmental awareness and encourage sustainable living practices.

## 🚀 Features

- User authentication system
- Interactive dashboard
- Modern responsive UI
- Sustainable living awareness platform
- Fast and clean frontend experience

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
